# SignalArc Contracts

This directory will contain the SignalArc Solidity / Foundry contract workspace.

Phase 6 starts with a minimal contracts and settlement prototype.

Current boundary:
- No production custody assumptions.
- No production settlement assumptions.
- No claim flow until explicitly implemented and tested.
- No Arc mainnet deployment.
- No private keys or environment files committed.

Planned structure:
- `src/` for Solidity contracts.
- `test/` for Foundry tests.
- `script/` for deployment or interaction scripts.
